# Legacy Security Tools

Archived security analysis tools from the infrastructure team.

## Tools Included

- **analyzer.py** - File hash analyzer for security verification

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python analyzer.py <target_file>
```

This will output MD5 and SHA256 hashes of the target file.

## Example

```bash
python analyzer.py suspicious_file.exe
```
